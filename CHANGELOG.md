# Changelog

All notable changes to qc2 will be documented in this file.

The project follows [Semantic Versioning](https://semver.org/). Until `v1.0.0`, minor releases may include breaking changes.

## [Unreleased]

### Added

- Standalone `cpwd` binary and bundled `qc2 cpwd` command.
- `qc2 list`, `qc2 help`, and `qc2 version` built-in commands.
- Cross-platform clipboard support for macOS, Windows, and Linux.
- Prebuilt release workflow with SHA-256 checksums.
- Installation scripts that verify downloaded release archives.
